## Deploying with GitHub Pages
1. Create a new repository on GitHub (e.g., `portfolio`).
2. Upload/push the contents of this folder to the repo (keep the `docs/` folder as-is).
3. Go to **Settings > Pages**.
4. Under **Build and deployment > Source**, choose **Deploy from a branch**.
5. Branch: `main`, Folder: `/docs`. Click **Save**.
6. Your site will be live at `https://<your-username>.github.io/<repo-name>/` within a minute or two.
